# NestMate

**NestMate** — веб-застосунок для пошуку співмешканців за сумісністю. Замість хаотичних оголошень на дошках, NestMate пропонує підбір на основі анкети сумісності: графік сну, ставлення до гостей, домашніх тварин, чистоти та інших важливих звичок побуту.

---

## Функціональність

- **Реєстрація та автентифікація** — захищений вхід через JWT-токени, хешування паролів
- **Анкета сумісності** — детальний профіль з уподобаннями та звичками
- **Розумний пошук** — фільтрація потенційних співмешканців за містом, бюджетом та відсотком сумісності
- **Алгоритм сумісності** — математичний розрахунок відсотка збігу профілів з детальною розбивкою по категоріях
- **AI-аналіз (Google Gemini)** — генерація персоналізованих текстових висновків щодо спільних інтересів та можливих конфліктів
- **Групи для оренди** — можливість створювати об'єднання користувачів для пошуку багатокімнатних квартир та подавати заявки на вступ
- **Перегляд профілів** — детальна картка кожного користувача у модальному вікні з усіма параметрами
- **Тестування** — перевірка системи на рівні бекенду (модульні та інтеграційні тести) та фронтенду (тестування UI-компонентів)

## Технічний стек

### Backend

| Технологія | Призначення |
|---|---|
| **FastAPI** | Асинхронний веб-фреймворк, REST API |
| **SQLAlchemy** | ORM для роботи з базою даних |
| **SQLite / PostgreSQL** | База даних |
| **Pydantic** | Валідація та серіалізація даних |
| **python-jose + passlib** | JWT-автентифікація, хешування паролів |
| **google-genai** | Інтеграція зі штучним інтелектом |
| **Pytest** | Модульне та інтеграційне тестування |
| **Uvicorn** | ASGI-сервер |

### Frontend

| Технологія | Призначення |
|---|---|
| **React 18** | SPA-інтерфейс |
| **Vite** | Збірка проєкту та сервер розробки |
| **Axios** | HTTP-запити до API |
| **Vitest + React Testing Library** | Тестування компонентів та інтерактивності |

## Запуск локально

### Передумови

- Python 3.12+
- Node.js 18+

### 1. Клонування репозиторію

```bash
git clone https://github.com/erisfelixx/nestmate-project.git
cd nestmate-project
```

### 2. Налаштування бекенду

```bash
cd backend

# Створити та активувати віртуальне середовище
python -m venv venv
source venv/bin/activate        # Linux/macOS
# або
venv\Scripts\activate           # Windows

# Встановити залежності
pip install -r requirements.txt
```

Створіть файл `.env` у папці `backend/`:

```env
DATABASE_URL=sqlite:///./nestmate.db
SECRET_KEY=your_secret_key_here
ALGORITHM=HS256
ACCESS_TOKEN_EXPIRE_MINUTES=30
GEMINI_API_KEY=your_google_gemini_api_key_here
```

Запустити тести бекенду:

```bash
pytest -v
```

Запустити сервер:

```bash
uvicorn app.main:app --reload
```

API буде доступне за адресою: `http://localhost:8000`  
Документація Swagger: `http://localhost:8000/docs`

### 3. Налаштування фронтенду

Відкрийте новий термінал.

```bash
cd frontend
npm install
```

Запустити тести фронтенду:

```bash
npm run test
```

Запустити застосунок:

```bash
npm run dev
```

Застосунок буде доступний за адресою: `http://localhost:5173`

## API

Основні ендпоінти:

| Метод | Шлях | Опис |
|---|---|---|
| `POST` | `/auth/register` | Реєстрація нового користувача |
| `POST` | `/auth/login` | Вхід, отримання JWT-токена |
| `GET` | `/profiles/me` | Профіль поточного користувача |
| `PUT` | `/profiles/me` | Оновлення профілю та анкети |
| `GET` | `/matches/` | Список сумісних співмешканців (відсортований) |
| `GET` | `/matches/analyze/{id}` | Отримання AI-аналізу сумісності з користувачем |
| `POST` | `/groups/` | Створення нової групи для оренди |
| `POST` | `/groups/{id}/apply` | Подача заявки на вступ до групи |

Повна інтерактивна документація доступна на `/docs` (Swagger UI) після запуску сервера.
